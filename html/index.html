Testing docker with nginx

some commands

  1. docker run -p 8080:80 -v (volum) $/file -> run docker and save share you app to not lost data
  2. docker-compose up -d -> docker-compose runs automatically all services
  3. docker ps -> see info about containers (id, image, command, port...)
  4. docker exec 'container name' ls -> list directories (inside container)
  5. docker exec -it (interactivy mode) 'container name' bash -> get into container
  6. cat index.html -> list html of nginx (or other, if it has)
  7. rm -f index.html -> remove file
  8. echo "hello world" > index.html -> create file
  9. docker build -t 'dockerhubname'/'project-name':latest -> run build
  10. docker push 'image-name' -> share image on dockerhub - allow others to experiment it.
